**什么是SPYDL1D**

SPYDLID系列指标是DOF Studio旗下DOF Think Tank所领导的Project Theia金融计划的初始核心产物。</br>
Project Theia是一个公开性计划，旨在为最广泛人群提供公开免费的投资策略与投资科普，以实现投资人类共同未来、积累人类共同财富的目的。</br>
Project Theia所公开的成果包含一系列金融指标，由于金融指标普遍具有时限性，SPYDLID指标将用版本号区分指标的更新。</br>
例如，截至目前，本文中使用的指标模型版本为SPYDLID01（构建日期：2025-06-21）。

**更新**

SPYDL1D现在由程序自动计算和自动更新，将每日盘内不定时和盘后计算指标值。</br>
您现在可以通过GitHub Pages的方式实时访问该指标。</br>
团队认为，在自动化的更新下，指标的更新延迟能够缩减到15分钟以内。
